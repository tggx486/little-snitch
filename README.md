## Little Snitch 是什么？能干啥？

简单说，Little Snitch 是 macOS 上的应用级防火墙。系统自带的防火墙只管进来的流量，而 Little Snitch 专注于**出站连接**——哪个 App 想往外发数据，必须先过它这关。

它在 macOS 上已经有超过二十年历史，算是这个领域没什么争议的老大哥。第六代更新之后，界面更清爽，功能也更完整。

主要能做这些事：

**实时连接地图**：可视化展示你的 Mac 正在和全球哪些服务器通信，服务器位置、端口、协议一眼看清。

**连接警报**：任何软件试图联网，弹窗问你放不放行。你来决定，不是软件来决定。

**静默模式**：如果不想被打扰，可以切到静默模式，先让连接过去，稍后再统一审查和制定规则。

**黑名单订阅**：内置丰富的订阅源，可以直接订阅专业用户维护的规则，一键屏蔽广告服务器、追踪器、恶意站点。

**DNS 加密**：支持 DoH（DNS over HTTPS）和 DoT（DNS over TLS），DNS 请求不再裸奔，防止被运营商或中间人窃听篡改。

**菜单栏图标**：随时查看实时上下行流量，快速切换控制模式，不用每次都打开主窗口。

**按软件/域名分组分析**：历史连接数据可以长期保存，按 App 或域名分组浏览，还支持高级搜索筛选。
